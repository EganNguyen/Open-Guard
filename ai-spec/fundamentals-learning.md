# 📘 Software Fundamentals — Study Guide

## 📌 Overview
Software fundamentals are the core concepts required to design, build, and maintain reliable software systems. Mastering these ensures strong problem-solving and system design skills.

---

# 1. 🧠 Algorithms & Logic

## What is an Algorithm?
A step-by-step procedure to solve a problem.

## Key Concepts
- Time Complexity (Big-O)
- Space Complexity
- Optimization

## Common Examples
- Sorting (QuickSort, MergeSort)
- Searching (Binary Search)

---

# 2. 🗂️ Data Structures

## Purpose
Organize and store data efficiently.

## Core Structures
- Array
- Linked List
- Stack (LIFO)
- Queue (FIFO)
- Tree (Binary Tree, BST)
- Graph

## Key Idea
> Choosing the right data structure = better performance

---

# 3. 💻 Programming Fundamentals

## Basics
- Variables & Data Types
- Control Flow (if/else, loops)
- Functions

## Core Paradigms
- Procedural Programming
- Object-Oriented Programming (OOP)

## OOP Principles
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

---

# 4. 🧩 Software Design Principles

## Clean Code Principles
- DRY — Don't Repeat Yourself
- KISS — Keep It Simple
- YAGNI — You Aren’t Gonna Need It

## SOLID Principles
- S — Single Responsibility
- O — Open/Closed
- L — Liskov Substitution
- I — Interface Segregation
- D — Dependency Inversion

# 🎯 Study Tips

- Focus on **understanding**, not memorizing
- Practice with real problems (LeetCode, projects)
- Build small systems to connect concepts
- Review regularly
