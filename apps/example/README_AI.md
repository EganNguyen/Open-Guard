# Connected Example App (React)

**Core Intent:** A demo React application demonstrating SDK usage and "Fail-Closed" security.
**Structure:**
- `server/index.ts`: Backend Express proxy using the SDK.
- `client/`: React frontend (Vite).

**AI Rules:**
- Keep changes minimal.
- Use `apps/example/server/routes.ts` for new endpoints instead of dumping into `index.ts`.
